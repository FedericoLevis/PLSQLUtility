# PA_UTL Package
PLSQL Utility Package :   
  - **LOG**: log, log_elapsed_sec, log_elapsed_msec are used to log with various level (LOG_LEV_ERR,..,LOG_LEV_TRACE) into Table UTL_LOG  
  - **OTHER Utility** Function: time_get_diff_sec, bool_to_varchar2,...
