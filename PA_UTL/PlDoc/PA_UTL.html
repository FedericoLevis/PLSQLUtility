log 

Procedure log(p_loglev  IN NUMBER,
              p_feature IN VARCHAR2,
              p_sp      IN VARCHAR2,
              p_logmsg  IN VARCHAR2,
              p_logId   IN NUMBER DEFAULT NULL,
              p_dbmsOut IN BOOLEAN DEFAULT FALSE)

LOG a Message into UTL_LOG NOTES:
 - log is done only if p_loglev is enabled in UTL_CFG for p_feature
 - The Aging of UTL_CFG is automatically performed by this routine, calling locale procedure LOG_AGING that uses UTL_LOG_CFG configuration

Parameters
p_loglev   IN  NUMBER     LOG_LEV_xx: 0=LOG_LEV_ERR  1= LOG_LEV_WARN , .....
p_feature   IN  VARCHAR2   Calling Feature
p_sp   IN  VARCHAR2   Calling SP (SP of the Feature p_feature)
p_logmsg   IN  VARCHAR2   Message to Log
[p_logId]   IN  VARCHAR2   Optional ID used to group together the log messages of the same CALL
[p_dbmsOut]   IN BOOLEAN     default FALSE. TRUE to call also DMBS_OUTPUT.PUT_LINE - if set to TRUE you can set DBMS_OUTPUT.ENABLE (buffer_size => NULL)
